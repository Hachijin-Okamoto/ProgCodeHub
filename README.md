# ProgCodeHub
- プログラミングの教育現場で、問題とそれに対して提出されたコードを閲覧するアプリ
- REST API / GraphQLの性能比較用アプリでもあります。

## 実行方法
```bash
$ npm install
$ npm run prisma:generate
$ npm run build
$ npm start
```
`http://localhost:8000/ui/rest`か`http://localhost:8000/ui/graphql`にアクセスすることで、本アプリを体験いただけます。
